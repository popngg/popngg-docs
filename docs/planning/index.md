## 핵심 방향

- High☆Cheers 변경 사항을 우선 지원합니다.
- 빠른 MVP 출시 후 기능을 확장합니다.
- 프론트에서 하던 데이터 가공은 가능한 한 백엔드 API 응답으로 이동합니다.
- 추후 오픈소스 공개를 고려해 문서와 설계를 공개 가능한 형태로 유지합니다.
