# 프로젝트 개요

## 목적

popn.gg는 pop'n music 플레이어가 곡, 채보, 플레이데이터, 팝클, 랭킹 정보를 확인할 수 있는 서비스입니다.

이번 리팩토링의 목적은 기존 백엔드의 구조를 정리하고, High☆Cheers 기준의 게임 시스템 변경을 반영하며, MVP 출시 후 확장 가능한 기반을 만드는 것입니다.

## 배경

기존 백엔드는 `chart` 테이블에 곡 메타데이터와 채보 메타데이터가 함께 저장되어 있었습니다. High☆Cheers에서 장르명 표기가 부활하고 랭크 체계가 변경되면서, 기존 구조로는 데이터 식별과 표시 정책을 안정적으로 유지하기 어렵습니다.

## 목표

- `song`과 `chart`를 분리합니다.
- High☆Cheers 랭크와 난이도 표시를 지원합니다.
- 플레이데이터 갱신과 랭킹 조회를 안정화합니다.
- 이메일 기반 비밀번호 복구를 제공합니다.
- Jenkins, Docker, Flyway 기반 운영 흐름을 준비합니다.

## 비목표

- MVP에서 모든 검색 태그/기여 기능을 구현하지 않습니다.
- MVP에서 모든 운영 모니터링 자동화를 완료하지 않습니다.
- MVP에서 상수 정책 관리 UI를 만들지 않습니다.
