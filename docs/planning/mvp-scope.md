# MVP 범위

## 포함

- 유저 등록/갱신/로그인
- 이메일 기반 비밀번호 복구
- 곡/채보 조회
- 플레이데이터 갱신
- 유저별 전체 플레이데이터 조회
- 유저 팝클 테이블
- 곡별 랭킹
- 레벨별 랭크/메달 집계
- 갱신 로그
- 플레이데이터 히스토리
- Jenkins + Docker 배포
- Flyway schema migration

## MVP 이후

- 검색 태그 기여
- 한국어 검색 태그 승인 플로우
- 이메일 인증 강화
- 상세 모니터링
- 관리자 데이터 검수 UI
- 상수 정책 DB 관리 UI
- strict judgement/gauge 데이터 확장

## 출시 전략

1. 기존 DB 구조를 기준으로 데이터 이전 계획을 확정합니다.
2. Flyway로 신규 스키마를 생성합니다.
3. 기존 데이터를 신규 스키마로 이전하고 검증합니다.
4. API를 MVP 범위로 먼저 공개합니다.
5. 운영 중 필요한 기능을 작은 단위로 확장합니다.
